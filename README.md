# ClientsManager

Hybrid app develop with Ionic 3 that will manage clients of health insurance broker. The app will manage the payment history, the insurance selected and it amount, the amount paid each month foreach client 

## Getting Started

The project is shared in github repository: https://github.com/gerazaporta/clients-manager

```
git clone https://github.com/gerazaporta/clients-manager
```


### Prerequisites

NodeJS must be installed and configured, a LTS version would be the best option to guarantie complete and long term functionalities. Ionic cli should be installed too

### Installing

For install NodeJS an installer may be downloaded from [Node JS](https://nodejs.org/en/) official website, depending of your OS. Ionic cli must be installed too following indications in [Ionic framework](https://ionicframework.com/docs/).  

After get installed NodeJS in command line tool the version of can be checked:
```
$ node -v
v10.13.0
```
Ionic can be installed from command line tool using NPM with install command:

```
$ npm install -g ionic cordova

```
